# Activity Diagrams

## AD-01 Translation Workflow
```mermaid
flowchart TD
    A([Start]) --> B[User inputs text]
    B --> C{Text empty?}
    C -- Yes --> Z([End])
    C -- No --> D[Run translate]
    D --> E{Source = auto?}
    E -- Yes --> F[Detect language]
    E -- No --> G[Use selected source]
    F --> H[Try phrase translation]
    G --> H
    H --> I{Phrase found?}
    I -- Yes --> J[Set phrase result]
    I -- No --> K[Try word translation]
    K --> L{Word found?}
    L -- Yes --> M[Set word result]
    L -- No --> N[Run sentence word-by-word translation]
    J --> O[Get word details if available]
    M --> O
    N --> O
    O --> P[Add input to history]
    P --> Q[Update UI output]
    Q --> Z
```

## AD-02 Voice Input Workflow
```mermaid
flowchart TD
    A([Start]) --> B[User taps mic]
    B --> C{Speech recognition supported?}
    C -- No --> D[Show unsupported message] --> Z([End])
    C -- Yes --> E{Currently listening?}
    E -- Yes --> F[Stop recognition] --> Z
    E -- No --> G[Configure recognition lang]
    G --> H[Start recognition]
    H --> I[Receive transcript]
    I --> J[Set transcript to input]
    J --> K[Auto-run translate]
    K --> L[Update details and history]
    L --> Z
```

## AD-03 Favorite Toggle Workflow
```mermaid
flowchart TD
    A([Start]) --> B[User clicks favorite]
    B --> C{Already favorite?}
    C -- Yes --> D[Remove from list]
    C -- No --> E[Add to list]
    D --> F[Persist localStorage]
    E --> F
    F --> G[Render updated state]
    G --> Z([End])
```

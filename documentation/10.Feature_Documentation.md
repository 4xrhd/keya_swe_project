# Feature Documentation

## Core Features

### 1. Hybrid Multilingual Translation
- **Offline Mode:** Uses the high-performance local dictionary dataset containing comprehensive word details for hundreds of core vocabulary terms.
- **Online Fallback:** If a queried word or full sentence isn't found offline, the application seamlessly triggers a request to the free, no-key-required **MyMemory Translation API** with a 5-second `AbortController` timeout for instant, high-quality results.
- **Visual Indicators:** Displays a premium `"Translated via API"` badge next to target fields when fallbacks are successfully triggered, keeping the user fully informed.

### 2. Intelligent English Lemmatizer & Stemmer
- **Suffix Removal:** Dynamically handles plurals (`-s`, `-es`, `-ies`), past tenses (`-ed`), and gerunds (`-ing`) to resolve base lemmas offline.
- **Irregular Verb Mapper:** Contains a built-in irregular conversion dictionary mapping inflections (e.g., `"done"`, `"did"`, `"does"`, `"doing"` $\rightarrow$ `"do"`; `"went"`, `"gone"`, `"going"` $\rightarrow$ `"go"`) back to their primary definitions.
- **High Compatibility:** Guarantees that search queries do not require exact, rigid string matches to retrieve offline definitions.

### 3. Autocomplete Search
- Suggests words/phrases dynamically as the user types.
- Runs cross-language matches in both English and all translated forms.
- Leverages cached reverse indexes for instant, low-latency rendering.

### 4. Word Detail Panel
- Renders comprehensive entries including parts of speech, phonetic spellings, definitions, real-world examples, synonyms, and antonyms.
- Includes a language mapping sub-panel to see the word translated into all 11 target languages simultaneously.

### 5. Favorites & Search History
- **Favorites:** Easily bookmark target entries with a one-click toggle.
- **History:** Automatically tracks up to 50 recent translation operations.
- **Persistence:** Uses browser `localStorage` under keys `dictionary_favorites` and `dictionary_history` for immediate load-on-startup.

### 6. Voice Input (Speech-to-Text)
- Utilizes the browser's native `SpeechRecognition` API.
- Converts speech into structured text on-the-fly and automatically triggers a translation.

### 7. Text-to-Speech (Pronunciation Helper)
- Integrates browser `SpeechSynthesis` API.
- Implements smart **BCP-47 language matching** to ensure appropriate localized accents are applied (e.g. US/UK English, Spanish, Hindi, German, French, Arabic, Chinese).

### 8. Quality-of-Life Tools
- **Language Swapping:** One-click swapping of source and target languages.
- **Random Word Generator:** Instantly loads a random comprehensive word to encourage learning.
- **Copy-to-Clipboard:** Quick copy buttons on both input and output panels.
- **Clear Settings:** Instant button to reset history.

---

## Known Constraints
- **Offline Scope:** Complex sentences or non-standard words default to online APIs. If offline *and* a word is missing, it falls back to word-by-word structural sentence layout.
- **Browser Compatibility:** Web Speech APIs depend on modern browser compatibility. On unsupported browsers, voice options gracefully hide.

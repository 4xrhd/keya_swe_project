# Feature Documentation

## Core Features
1. **Multilingual Translation**
- Supports direct word translation, phrase translation, and sentence fallback translation.
- Source language can be selected or auto-detected.

2. **Autocomplete Search**
- Suggests words/phrases from English and translated forms.
- Includes prefix and contains matching.

3. **Word Detail Panel**
- Shows part of speech, phonetic, definitions, examples, synonyms, antonyms, and all language mappings.

4. **Favorites**
- Toggle and persist favorite words.
- Quickly reopen favorite entries.

5. **Search History**
- Stores up to 50 recent entries.
- Supports full clear action.

6. **Voice Input**
- Uses Speech Recognition API for transcript input.
- Automatically translates recognized text.

7. **Text-to-Speech**
- Speaks source/translated text with language-aware voice selection.

8. **Quality-of-Life Tools**
- Swap source/target languages.
- Random word generator.
- One-click copy for output text.

## Known Constraints
- Translation quality is limited to local dataset coverage.
- No user authentication or cloud sync in current build.
- Speech APIs are browser-dependent.
